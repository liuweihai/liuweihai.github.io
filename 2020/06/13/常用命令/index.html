    key描述示例
    pull下载指定进行docker pull [镜像名称]
    save保存已有镜像docker save [镜像名称] > [压缩名称].tar
    load解压当前已有镜像docker load < [压缩名称].tar
    tag复制已有镜像（类似创建快捷方式）docker tag [镜像名称] [新的引用名称]
    tag引用已有镜像（类似创建快捷方式）docker tag [镜像名称] [新的镜像名称]
    rmi删除镜像docker rmi [镜像名称] (默认删除原镜像)
    run运行某个镜像docker run -d –name [启动名称] [镜像名称] [镜像名称]
    logs打印某个镜像日志docker logs -f
    kill杀掉某个镜像docker kill [任务name]
    ps列出所有进程docker ps
    stop停止已启动的进程docker stop [任务name]
    rm删除任务名称docker rm -f [任务name]
    exec进入容器docker exec -it [容器id]
    inspect查看容器详细信息docker inspect [CONTAINER ID]
    info查询容器属性信息docker info
    cp将宿主机文件拷贝到容器中docker cp $(pwd)/[文件] [任务name]:容器中地址
    commit创建新到镜像docker commit [镜像name] [新的镜像名称]
    container网络模式docker run –name=[启动名称] –net=container:conan [镜像名称]
